# repofleet v1.1.0

## What's new

- **Startup update check** — on launch, `fleet` asks GitHub for the latest published release in the background. If your build is behind, the header shows a colored hint: current version in **yellow** `~>` latest in **green** (e.g. `v1.0.1 ~> v1.1.0`). The check is silent on failure (offline is fine) and never nags `dev` builds or an up-to-date binary.

## Install / upgrade

```bash
go install github.com/dutraph/repofleet/cmd/fleet@latest
# or
curl -fsSL https://raw.githubusercontent.com/dutraph/repofleet/main/install.sh | bash
# or grab a prebuilt binary from the assets below (verify against SHA256SUMS)
```
