# repofleet v1.0.2

## What's new

- **Filter-aware counts** — when a provider filter (`t`) or search (`/`) is active, the header counts now reflect the visible subset instead of the totals. The repo list shows `repos · N found · M dup · <type>`, and the duplicates screen (`D`) shows `duplicates · N groups · <type>` for the filtered set.

## Housekeeping

- Release notes are now organized under `releases/` (one file per version, with an index).

## Install

```bash
go install github.com/dutraph/repofleet/cmd/fleet@latest
# or
curl -fsSL https://raw.githubusercontent.com/dutraph/repofleet/main/install.sh | bash
# or grab a prebuilt binary from the assets below (verify against SHA256SUMS)
```
